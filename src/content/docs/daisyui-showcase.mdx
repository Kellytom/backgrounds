## Code Integration

Here's how to add DaisyUI to your project:

```bash
# Install DaisyUI
npm i -D daisyui@latest

# Add to your Tailwind config
module.exports = {
  plugins: [require("daisyui")],
}
```

```html
<!-- Use components in your HTML -->
<button class="btn btn-primary">Primary Button</button>
<div class="card bg-base-100 shadow-xl">
  <div class="card-body">
    <h2 class="card-title">Card Title</h2>
    <p>Card content goes here</p>
  </div>
</div>
```

## Theme Integration

DaisyUI works perfectly with our Hacker1 theme system:

```css
/* Custom DaisyUI theme variables */
[data-theme="dark"] {
  --p: 259 94% 51%;
  --pc: 259 96% 91%;
  --s: 314 100% 47%;
  --sc: 314 100% 91%;
  --a: 174 60% 51%;
  --ac: 174 60% 11%;
}
```
